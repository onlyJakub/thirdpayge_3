Co znaczy skrót ID? ID - Identyfikator.
Jaki jest cel stosowania ID? Stosowanie identyfikatora (ID) jest kluczowe w celu odróżnienia podobnych elementów, co umożliwia lepszą organizację strony internetowej.
Jakie jest znaczenie znaku #? Znak # poprzedza definicję identyfikatora, który jest używany do jednoznacznego identyfikowania elementu w CSS.
Jak następuje wywołanie ID (wraz z wytłumaczeniem)? Wywołanie identyfikatora odbywa się za pomocą id="nazwa zdefiniowanego w CSS identyfikatora". Składa się z selektora i nazwy poprzedzonej znakiem #.
Uwagi na temat stosowania ID (trzy) konieczne wyliczenia.
  • Różnica między id a class: Identyfikator może wystąpić tylko raz w całym dokumencie, podczas gdy klasę można powielać.
  • Podobnie jak klasy, identyfikatory można używać wielokrotnie, choć jest to błędne zachowanie.
  • Elementy oznaczone ID muszą być unikatowe na stronie, dlatego lepiej używać 'class' w przypadku wielokrotnego występowania.
