1. Wymienić oraz opisz dwie części selektora,


    2. Selektor w CSS składa się z dwóch głównych części: elementów docelowych, które określają na jakim elemencie HTML będą stosowane style (np. h1 dla nagłówków), oraz klas lub identyfikatorów (np. .klasa lub #identyfikator), które dodatkowo precyzują, które elementy na stronie mają zastosować konkretne style.

  2. Znaczenie nawisów klamrowym {...},


    3. Nawiasy klamrowe ({}) są używane do wstawian fragmentu kodu. Wewnątrz nich umieszczamy blok kodu.

  3. Opisz cechy,


    4. Selektor w CSS to narzędzie, które identyfikuje elementy na stronie do stylowania, uwzględniając specyficzność, możliwość zastosowania wielu warunków, unikalność lub powtarzalność oraz działanie kaskadowe.


  4. Opisz wartości


    5. Wartości w selektorze w CSS to elementy HTML, klasy, identyfikatory oraz pseudoelementy i pseudoklasy, które precyzyjnie określają, które elementy na stronie zostaną objęte określonymi stylami CSS.

  5. Kiedy stosujemy znak średnika (;), kiedy nie jest konieczny


    6. Znak średnika (;) w CSS jest używany do oddzielenia deklaracji stylu od siebie. Jest on konieczny po każdej deklaracji stylu, aby oznaczyć jej koniec.
